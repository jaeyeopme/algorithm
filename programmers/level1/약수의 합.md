# 약수의 합

[문제 링크](https://school.programmers.co.kr/learn/courses/30/lessons/12928)

```java
class Solution {

    public int solution(int n) {
        if (n < 2) {
            return n;
        }

        double s = Math.sqrt(n);
        int answer = 0;

        for (int i = 1; i <= s; i++) {
            if (n % i == 0) {
                if (i != s) {
                    answer += n / i;
                }
                answer += i;
            }
        }

        return answer;
    }

}
```
